## 七层网络协议

+ **应用层**
  + 如QQ等，其协议由自己定，HTTP协议，DNS（域名解析）协议,FTP（文件传输）协议,SMTP（邮件传输）协议等
+ **表示层**
  + 加密，格式转换
+ **会话层**
  + 解除或者建立与其他节点的联系
+ **传输层**
  + 交换机之类,TCP协议，UDP协议等
+ **网络层**
  + 你所在的网络节点，IP(网络层)协议等
+ **数据链路层**
  + APM协议，FDDI协议等
  + 分为逻辑链路子层和媒体访问控制子层（MAC）
+ **物理层**
  + OSI的物理层规范（如电流多大）

A --> B的传输是应用层到物理层层层加密的过程，那么B在拿到数据时就是从物理层到应用层层层解密的过程，顺序不能错误

## 五层网络协议

 五层网络协议是将七层网络协议中的表示层和会话层合并到了应用层
